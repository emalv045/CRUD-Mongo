# CRUD-Mongo
CRUD with MongoDB and Node JS

Sistema de Renta de Vehiculos que contiene CRUD de:

	Clientes
	Vehiculos
	Rentas

Además permite consultar vehículos según filtros de Placa,Marca,Rango de Precio,entre otros.
